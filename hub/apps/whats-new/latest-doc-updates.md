# <a name="latest-updates-to-the-windows-developer-docs"></a>Neueste Aktualisierungen an der Dokumentation für Windows-Entwickler

Die Dokumentation für Windows-Entwickler wird regelmäßig mit neuen und verbesserten Informationen und Inhalten aktualisiert. Hier ist eine Zusammenfassung der Änderungen zum 3. September 2020.

Hinweis: Eine spezifische Liste der APIs, die als Teil von Windows 10 Build 19041 (auch als 2004 bezeichnet) hinzugefügt wurden, finden Sie in [dieser Liste](/windows/uwp/whats-new/windows-10-build-19041-api-diff).


## <a name="news"></a>Neuigkeiten

In diesem Monat haben wir mehrere tausend absolute Links in relative Links übersetzt, um die Navigation zu beschleunigen und die Offlinebuch-Umgebung zu verbessern. Bitte informieren Sie uns, wenn Sie auf beschädigte Links stoßen. Sie können von den meisten Seiten aus Feedback senden oder erreichen uns über Twitter mit unserem Handle **WindowsDocs**.

Zu den Highlights dieses Monats gehören:

* Aktualisierungen einer 4-teiligen Tutorial-Reihe zur Verwendung von WinUI-[Benutzeroberfläche](https://docs.microsoft.com/windows/uwp/design/basics/xaml-basics-ui) und -[Datenbindung](https://docs.microsoft.com/windows/uwp/data-binding/xaml-basics-data-binding) sowie [Beispielcode](https://github.com/microsoft/Windows-appsample-photo-lab/tree/master/xaml-basics-starting-points). 

* [Windows-Terminal 1.2](https://docs.microsoft.com/windows/terminal/) und Vorschau 1.3 – Aktualisierungen der Dokumentation.

* Neue Seite [Einrichten der Entwicklungsumgebung](https://docs.microsoft.com/windows/dev-environment/overview).

* Einige neue FAQs und Aktualisierungen zu [KI- und Machine Learning](https://docs.microsoft.com/windows/win32/direct3d12/gpu-faq).

* Ein neues WinRT/C++-Thema über [Asynchronität und Interoperabilität zwischen C++/WinRT und C++/CX](https://docs.microsoft.com/windows/uwp/cpp-and-winrt-apis/interop-winrt-cx-async).
