# <a name="latest-updates-to-the-windows-developer-docs"></a>Neueste Aktualisierungen an der Dokumentation für Windows-Entwickler

Die Dokumentation für Windows-Entwickler wird regelmäßig mit neuen und verbesserten Informationen und Inhalten aktualisiert. Die nachfolgend aufgeführten Dokumente gehören zu denjenigen, die in letzter Zeit umfassend überarbeitet wurden.

Unter anderem wurden diese Themen vor Kurzem umfassend aktualisiert (bis 2. Juni 2020). 

Hinweis: Mit der Veröffentlichung von Windows 10 Build 19041 und einigen systematischen Umstrukturierungen der Dokumentation ist die Liste in diesem Monat *sehr* lang. Eine spezielle Liste der APIs, die als Teil von Windows 10 Build 19041 hinzugefügt wurden, finden Sie in [dieser Liste](https://docs.microsoft.com/windows/uwp/whats-new/windows-10-build-19041-api-diff).
