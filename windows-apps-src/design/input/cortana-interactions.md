# <a name="cortana-interactions-in-windows-apps"></a>Cortana-Interaktionen in Windows-apps

Cortana bietet ein stabiles und umfassendes Erweiterungsframework, mit dem Sie Funktionen aus Ihrer App oder einem Dienst nahtlos in die Cortana-Benutzeroberfläche integrieren können.

## <a name="weve-moved"></a>Wir sind umgezogen

Die gesamte Entwicklerdokumentation für Cortana-Features und -Dienste ist jetzt über [Cortana Dev Center](https://developer.microsoft.com/cortana) verfügbar.

Informationen zu den ersten Schritten finden Sie in der [Übersicht über das Cortana Skills Kit](/cortana/skills/overview).

Weitere Informationen zum Erweitern von Cortana mit Funktionalität aus einer Windows-App mithilfe von Sprachbefehlen finden Sie unter [Cortana Voice-Befehle](/cortana/voice-commands/vcd).
