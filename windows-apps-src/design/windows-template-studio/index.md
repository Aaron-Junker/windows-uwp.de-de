# <a name="windows-template-studio"></a>Windows Template Studio

![Herobild](images/wts1.png)

Windows Template Studio (WTS) ist eine Visual Studio 2017- und Visual Studio 2019-Erweiterung, die mit einer assistentenbasierten Oberfläche die Erstellung neuer UWP-Apps (Universelle Windows-Plattform) beschleunigt. Das resultierende UWP-Projekt ist wohlgeformter lesbarer Code, der die neuesten Windows 10-Features enthält und dabei bewährte Muster und bewährte Methoden implementiert. Der generierte Code enthält Links zu Dokumenten, Stack Overflow and Blogs mit nützlichen Informationen. WTS unterstützt die Erstellung von Apps in C# oder Visual Basic.

Weitere Informationen finden Sie im [Projektrepository auf GitHub](https://github.com/microsoft/windowsTemplateStudio).
