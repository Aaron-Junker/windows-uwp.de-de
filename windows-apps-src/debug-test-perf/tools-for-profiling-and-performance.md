# <a name="tools-for-profiling-and-performance"></a>Tools für Profilerstellung und Leistung


Microsoft bietet verschiedene Tools zur Verbesserung der Leistung Ihrer App für die universelle Windows-Plattform (UWP). Unter den folgenden Links finden Sie weitere Informationen zur Verwendung dieser Tools.

**Reaktionsfähigkeit von XAML-Benutzeroberflächen-Tools in Visual Studio**. Eines der zu verwendenden zur Messung der Auswirkung auf die Leistung Ihrer App bewährte Tools ist der XAML-UI Reaktionsfähigkeit Tool. Dies wurde [aktualisiert](http://blogs.msdn.com/b/wpf/archive/2015/01/14/new-ui-performance-analysis-tool-for-wpf-applications.aspx) , um noch mehr Szenarios zu unterstützen.
