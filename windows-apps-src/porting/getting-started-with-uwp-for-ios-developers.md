# <a name="getting-started-with-uwp-for-ios-developers"></a>Erste Schritte mit UWP für iOS-Entwickler


Wenn Sie ein IOS-Entwickler sind, der die Entwicklung für Windows 10 in Erwägung zieht, sind diese Dokumente ein guter Ausgangspunkt. Sie werden einige Konzepte vorstellen, die Sie kennen müssen, wenn Sie mit dem Schreiben von apps beginnen, und Sie erfahren, wie Sie Ihre Arbeit auf dem Microsoft Store veröffentlichen können.

In diesem Abschnitt erfahren Sie schrittweise, wie Sie mit Microsoft Visual Studio und der Programmiersprache C# eine einfache App erstellen und wie sich der Prozess vielleicht von Ihren derzeit verwendeten Tools unterscheidet. (Kein Fan von C#? Andere verfügbare Programmiersprachen und -Tools werden unter [Erste Schritte: Auswählen einer Programmiersprache](getting-started-choosing-a-programming-language.md) behandelt.)

Windows 10 bietet eine neue Plattform zum Erstellen ansprechender apps auf Desktop, Laptop, Tablet, Phone-Geräten und mehr. Da UWP-Apps viele einzigartige Funktionen bieten, führt die einfache Portierung einer iOS-App dazu, dass die Vorteile dieser Features nicht genutzt werden können. Es empfiehlt sich deshalb, die neuen Steuerelemente und Funktionen auszuprobieren. Sie werden sehen, dass sie die Entwicklungsarbeit erleichtern und die Gestaltung neuer Apps ermöglichen.

Fazit: Portieren Sie Ihre App nicht einfach nur, sondern **wandeln Sie sie um**, und profitieren Sie von den neuen Funktionen und neuen Geräten. Legen Sie nicht den kleinsten gemeinsamen Nenner fest, sondern erstellen Sie umfassende Erfahrungen, die eindeutige Windows 10-Features wie Live-Kacheln, Benachrichtigungen und die Interaktion mit Cortana nutzen.

Um mit diesen exemplarischen Vorgehensweisen zu beginnen, benötigen Sie einen Computer, auf dem sowohl Windows 10 als auch Microsoft Visual Studio installiert ist. Sie können diese aus den [Entwickler Downloads für die Programmierung von UWP-apps](https://developer.microsoft.com/windows/downloads)herunterladen. Sie haben keinen PC? Kein Problem: Sie können einen Mac verwenden. Informationen hierzu finden Sie unter [Installieren von Windows und der Entwicklertools auf dem Mac](setting-up-your-mac-with-windows-10.md).
