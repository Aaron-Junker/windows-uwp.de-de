<a id="ID4EO"></a>

 
## <a name="mediarecord"></a>MediaRecord
 
MediaRecord Objekt verfügt über den folgenden Spezifikationen.
 
| Mitglied| Typ| Beschreibung| 
| --- | --- | --- | 
| id| string| Der Bezeichner der Medien, die von der Bing-Katalog oder Anbieter-Katalog verwendet.| 
| idType| string| Wie den Media-Bezeichner interpretiert. Mögliche Werte sind "Bing" und "Provider".| 
| name| string| Lokalisierte Name für die Medieninhalte.|
