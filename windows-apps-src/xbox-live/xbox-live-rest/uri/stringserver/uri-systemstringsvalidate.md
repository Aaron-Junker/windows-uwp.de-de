# <a name="systemstringsvalidate"></a>/system/strings/validate
Greift auf ein Array von Zeichenfolgen für die Überprüfung. Die Domäne für diese URIs ist `client-strings.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Gültigen Methoden

[POST (/ System/Zeichenfolgen/validate)](uri-systemstringsvalidatepost.md)

&nbsp;&nbsp;Akzeptiert ein Array von Zeichenfolgen für die Überprüfung aus, und gibt ein Array von Ergebnissen gleicher Größe.
