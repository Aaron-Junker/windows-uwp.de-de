# <a name="mediamarketplaceidcontentrating"></a>/media/{marketplaceId}/contentRating
Die Einstufung des Inhalts-Zugriffstoken. Die Domäne für diese URIs ist `eds.xboxlive.com`.
 
  * [URI-Parameter](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI-Parameter
 
| Parameter| Typ| Beschreibung| 
| --- | --- | --- | 
| marketplaceId| string| Erforderlich. String-Wert, der vom die <b>Windows.Xbox.ApplicationModel.Store.Configuration.MarketplaceId</b>.| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>Gültigen Methoden

[Abrufen (/media/ {MarketplaceId} / ContentRating)](uri-medialocalecontentratingget.md)
