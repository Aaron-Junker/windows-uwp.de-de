# <a name="usersmeinventory"></a>/users/me/inventory
Greift auf den Satz von Inventur, die derzeit mit dem angegebenen Benutzer zugeordnet sind. Die Domäne für diese URIs ist `inventory.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Gültigen Methoden

[Abrufen (/ Benutzer/me/das Inventar)](uri-inventoryget.md)

&nbsp;&nbsp;Enthält den Satz von Inventur, die derzeit mit dem angegebenen Benutzer zurück an den Aufrufer zugeordnet sind.
