# <a name="mediamarketplaceidmetadatamediaitemtypes"></a>/media/{marketplaceId}/metadata/mediaItemTypes
Greift auf alle unterstützten MediaItemTypes für die angegebene EDS-Version. Die Domäne für diese URIs ist `eds.xboxlive.com`.
 
  * [URI-Parameter](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI-Parameter
 
| Parameter| Typ| Beschreibung| 
| --- | --- | --- | 
| marketplaceId| string| Erforderlich. String-Wert, der vom die <b>Windows.Xbox.ApplicationModel.Store.Configuration.MarketplaceId</b>.| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>Gültigen Methoden

[GET (/media/ {MarketplaceId} / Metadata/MediaItemTypes)](uri-medialocalemetadatamediaitemtypesget.md)

&nbsp;&nbsp;Listet alle unterstützten MediaItemTypes für die angegebene EDS-Version.
