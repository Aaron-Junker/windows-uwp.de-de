# <a name="usersxuidxuidfeedback"></a>/users/xuid({xuid})/feedback

## <a name="uri-parameters"></a>URI-Parameter
 
| Parameter| Typ| Beschreibung| 
| --- | --- | --- | 
| xuid| ulong| Xbox User ID (XUID) des Benutzers, der gemeldet wird.|
