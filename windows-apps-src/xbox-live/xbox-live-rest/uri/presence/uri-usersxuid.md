# <a name="usersxuidxuid"></a>/users/xuid({xuid})
Zugriff auf das Vorhandensein von einem anderen Benutzer oder Client. Die Domäne für diese URIs ist `userpresence.xboxlive.com`.
 
  * [URI-Parameter](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI-Parameter
 
| Parameter| Typ| Beschreibung| 
| --- | --- | --- | 
| xuid| 64-Bit-Ganzzahl ohne Vorzeichen| Xbox User ID (XUID) des Zielbenutzers.| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>Gültigen Methoden

[GET (/users/xuid({xuid}))](uri-usersxuidget.md)

&nbsp;&nbsp;Entdecken Sie das Vorhandensein von einem anderen Benutzer oder Client.
