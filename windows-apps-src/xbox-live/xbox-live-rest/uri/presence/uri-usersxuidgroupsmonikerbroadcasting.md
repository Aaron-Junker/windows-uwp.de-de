# <a name="usersxuidxuidgroupsmonikerbroadcasting"></a>/users/xuid({xuid})/groups/{moniker}/broadcasting
Greift im Zusammenhang mit der XUID, die im URI angezeigt wird des Datensatzes vorhanden Broadcasting Benutzer anhand der Gruppen-Moniker. Die Domäne für diese URIs ist `userpresence.xboxlive.com`.
 
  * [URI-Parameter](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI-Parameter
 
| Parameter| Typ| Beschreibung| 
| --- | --- | --- | 
| xuid| string| Xbox User ID (XUID) des Benutzers im Zusammenhang mit der XUIDs in der Gruppe.| 
| moniker| string| Zeichenfolge, die die Gruppe von Benutzern zu definieren. Der einzige akzeptierte Moniker derzeit ist die "People", mit dem Großbuchstaben "P".| 
  
<a id="ID4E4B"></a>

 
## <a name="valid-methods"></a>Gültigen Methoden

[Abrufen (/ Benutzer/Xuid ({Xuid}) /groups/ {Moniker} / Übertragung)](uri-usersxuidgroupsmonikerbroadcastingget.md)

&nbsp;&nbsp;Ruft die Anwesenheit Datensatz Broadcasting Benutzer anhand der Gruppen-Moniker im Zusammenhang mit der XUID, die im URI angezeigt wird.
