# <a name="authorization-types"></a>Authentifizierungstypen
 
Xbox Live Services verwendet eine Reihe von Autorisierungstypen mit Anforderungen. Dieses Thema beschreibt die standard-Autorisierung.
 
| Typ| Beschreibung| 
| --- | --- | 
| Benutzer | Aufrufer ist ein autorisierter Benutzer mit Xbox Live. | 
| Titel | Aufrufer handelt es sich um einen Titel auf einer bewährten Plattform.|
