# <a name="uwp-app-streaming-install"></a>Installieren von UWP-App-Streaming
Durch die Installation des Universelle Windows Plattform (UWP)-App-Streaming können Sie angeben, welche Teile der App der Microsoft Store zuerst herunterladen soll. Wenn die wichtigen Dateien der App zuerst heruntergeladen werden, können Benutzer die App starten und mit dieser interagieren, während der Rest noch im Hintergrund heruntergeladen wird. 

Um die UWP-App-Streaming-Installation zu verwenden, müssen Sie die Dateien in Ihrer App in Abschnitte unterteilen. Zu diesem Zweck müssen Sie eine Inhaltsgruppenzuordnung erstellen, d. h. eine XML-Datei, die in Ihrer App enthalten ist und mit der Sie die Downloadpriorität und -reihenfolge festlegen können. Weitere Informationen finden Sie im nachfolgend verknüpften Thema.

Eine vollständige Anleitung zum Hinzufügen der UWP-App-Streaming-Installation zu Ihrer UWP-App finden Sie in dieser [Blogserie](https://blogs.msdn.microsoft.com/appinstaller/2017/03/15/uwp-streaming-app-installation/).
