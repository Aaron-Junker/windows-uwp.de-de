# <a name="depth-and-stencil-buffers"></a>Tiefen- und Schablonenpuffer


Ein *Tiefenpuffer* speichert Tiefeninformationen, um zu steuern, welche Bereiche eines Polygons gerendert oder verborgen werden. Ein *Schablonenpuffer* wird Maskieren von Pixeln in einem Bild verwendet. So können Spezialeffekte erstellt werden (z. B. Stanzeffekte; Decaling; Transparenzen, Ausblenden und Verwischen; Außenlinien und Silhouetten und zweiseitige Schablonen).
